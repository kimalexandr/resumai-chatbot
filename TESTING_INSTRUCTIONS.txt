ИНСТРУКЦИЯ ПО ТЕСТИРОВАНИЮ RESUMAI CHATBOT
===============================================

ПРОБЛЕМЫ, КОТОРЫЕ БЫЛИ ИСПРАВЛЕНЫ:
1. ✅ OpenAI API интеграция - исправлена
2. ✅ Загрузка файлов - добавлена отладка
3. ✅ Чат функциональность - восстановлена
4. ✅ Обработка ошибок - улучшена
5. ✅ OTP код - вечный для тестирования

КАК ЗАПУСТИТЬ:
1. ДВАЖДЫ кликните на файл LAUNCH_FINAL_FIXED.bat
2. Дождитесь сообщения "Running on http://127.0.0.1:5000"
3. Откройте браузер и перейдите на http://localhost:5000

КАК ТЕСТИРОВАТЬ:

1. ВХОД В СИСТЕМУ:
   - Введите любой email (например: test@test.com)
   - Нажмите "Получить код"
   - Скопируйте код из терминала или alert
   - Вставьте код и нажмите "Войти"

2. ТЕСТ ЗАГРУЗКИ ФАЙЛА:
   - Нажмите "Выбрать файл"
   - Выберите test_resume.txt
   - Нажмите "Загрузить файл"
   - Должно появиться сообщение об успешной загрузке

3. ТЕСТ АНАЛИЗА ВАКАНСИИ:
   - Вставьте ссылку на вакансию HH.ru
   - Нажмите "Анализировать вакансию"
   - Должен появиться анализ

4. ТЕСТ АНАЛИЗА ТЕКСТА:
   - Введите текст резюме в поле
   - Нажмите "Анализировать текст"
   - Должен появиться анализ

5. ТЕСТ ЧАТА:
   - Введите сообщение в поле чата
   - Нажмите Enter или кнопку "Отправить"
   - Должен появиться ответ

ОТЛАДОЧНАЯ ИНФОРМАЦИЯ:
- В терминале будут видны все DEBUG сообщения
- Если что-то не работает, проверьте терминал
- Все ошибки будут показаны с подробностями

ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:
1. Проверьте терминал на наличие ошибок
2. Убедитесь, что приложение запущено на порту 5000
3. Проверьте консоль браузера (F12 -> Console)
4. Попробуйте перезапустить приложение

ПРИМЕЧАНИЕ:
Приложение работает без OpenAI API ключа, используя fallback ответы.
Для полного AI-анализа нужно настроить OPENAI_API_KEY в config.py
