НАСТРОЙКА DEEPSEEK API ДЛЯ RESUMAI
====================================

1. ПОЛУЧЕНИЕ API КЛЮЧА:
   - Зайдите на https://platform.deepseek.com/
   - Зарегистрируйтесь или войдите в аккаунт
   - Перейдите в раздел "API Keys"
   - Создайте новый API ключ
   - Скопируйте ключ

2. НАСТРОЙКА ПЕРЕМЕННЫХ ОКРУЖЕНИЯ:
   Создайте файл .env в корневой папке проекта:
   
   DEEPSEEK_API_KEY=ваш_ключ_здесь
   SECRET_KEY=любой_секретный_ключ
   
   ИЛИ установите переменную окружения:
   set DEEPSEEK_API_KEY=ваш_ключ_здесь

3. УСТАНОВКА ЗАВИСИМОСТЕЙ:
   pip install openai>=1.0.0
   pip install -r requirements.txt
   
   ИЛИ используйте bat файл:
   install_deepseek.bat

4. ЗАПУСК ПРИЛОЖЕНИЯ:
   python run.py
   
   ИЛИ используйте bat файл:
   RUN_APP.bat

5. ПРОВЕРКА РАБОТЫ:
   - Откройте http://localhost:5000
   - Попробуйте загрузить файл или ввести ссылку на вакансию
   - DeepSeek должен проанализировать и дать рекомендации

ПРИМЕЧАНИЯ:
- DeepSeek использует модель deepseek-chat (быстрая и качественная)
- API ключ DeepSeek бесплатный для начала
- Приложение автоматически использует DeepSeek вместо Qwen
- Поддерживается анализ резюме, вакансий и общий чат
- Используется OpenAI клиент версии 1.0.0+ для совместимости

ПРЕИМУЩЕСТВА DEEPSEEK:
- Высокое качество ответов
- Быстрая обработка
- Поддержка русского языка
- Бесплатный уровень использования
- Простая интеграция через OpenAI клиент
- Современный API (OpenAI 1.0.0+)
