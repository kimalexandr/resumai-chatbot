РЕШЕНИЕ ПРОБЛЕМЫ С БАЛАНСОМ DEEPSEEK
========================================

ОШИБКА: "Insufficient Balance" (код 402)

ПРОБЛЕМА:
У вас недостаточно средств на балансе DeepSeek аккаунта для выполнения API запросов.

РЕШЕНИЯ:

1. ПОПОЛНЕНИЕ БАЛАНСА (рекомендуется):
   - Зайдите на https://platform.deepseek.com/
   - Войдите в свой аккаунт
   - Перейдите в раздел "Billing" или "Account"
   - Пополните баланс (обычно минимальная сумма $5-10)
   - После пополнения API заработает автоматически

2. ПРОВЕРКА БЕСПЛАТНОГО УРОВНЯ:
   - DeepSeek предоставляет бесплатные токены новым пользователям
   - Проверьте, не истек ли бесплатный период
   - Возможно, нужно подтвердить email или телефон

3. АЛЬТЕРНАТИВНЫЕ РЕШЕНИЯ:
   - Используйте другой API ключ (если есть)
   - Создайте новый аккаунт для получения бесплатных токенов
   - Используйте тестовый режим приложения

4. ТЕСТОВЫЙ РЕЖИМ:
   - Приложение автоматически переключается в тестовый режим
   - Вы получите демо-ответы без использования API
   - Функциональность сохранена, но без AI-анализа

ПРОВЕРКА РАБОТЫ:
1. Пополните баланс на DeepSeek
2. Перезапустите приложение
3. Попробуйте отправить сообщение в чат
4. Должен появиться полноценный AI-ответ

ПРИМЕЧАНИЯ:
- Баланс пополняется мгновенно
- API ключ остается тем же
- Приложение автоматически определит доступность API
- Тестовый режим работает без ограничений

ЕСЛИ ПРОБЛЕМА СОХРАНЯЕТСЯ:
1. Проверьте правильность API ключа
2. Убедитесь, что баланс пополнен
3. Попробуйте создать новый API ключ
4. Обратитесь в поддержку DeepSeek
