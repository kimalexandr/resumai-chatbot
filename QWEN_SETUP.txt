НАСТРОЙКА QWEN (DASHSCOPE) ДЛЯ RESUMAI

1. ПОЛУЧЕНИЕ API КЛЮЧА:
   - Зайдите на https://dashscope.aliyun.com/
   - Зарегистрируйтесь или войдите в аккаунт
   - Перейдите в раздел "API Keys"
   - Создайте новый API ключ
   - Скопируйте ключ

2. НАСТРОЙКА ПЕРЕМЕННЫХ ОКРУЖЕНИЯ:
   Создайте файл .env в корневой папке проекта:
   
   DASHSCOPE_API_KEY=ваш_ключ_здесь
   SECRET_KEY=любой_секретный_ключ
   
   ИЛИ установите переменную окружения:
   set DASHSCOPE_API_KEY=ваш_ключ_здесь

3. УСТАНОВКА ЗАВИСИМОСТЕЙ:
   Дважды кликните на install_qwen.bat
   
   ИЛИ выполните в командной строке:
   pip install dashscope==1.14.0
   pip install -r requirements.txt

4. ЗАПУСК ПРИЛОЖЕНИЯ:
   Дважды кликните на RUN_APP.bat
   
   ИЛИ выполните:
   python run.py

5. ПРОВЕРКА РАБОТЫ:
   - Откройте http://localhost:5000
   - Попробуйте загрузить файл или ввести ссылку на вакансию
   - Qwen должен проанализировать и дать рекомендации

ПРИМЕЧАНИЯ:
- Qwen использует модель qwen-turbo (быстрая и качественная)
- API ключ DashScope бесплатный для начала
- Приложение автоматически переключится с OpenAI на Qwen
